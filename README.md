# Automotive Suspension Analysis

## Objective
Design and analyze an automotive suspension component under loading conditions.

## Tools Used

- SolidWorks
- ANSYS

## Methodology

- Designed CAD geometry
- Applied material properties
- Generated mesh
- Applied load conditions
- Performed structural analysis

## Results

Maximum Stress:
223.56 MPa (Structural Steel)

Maximum Deformation:
0.23847 mm (Magnesium Alloy)
